# Pandas CSV Practice Coach

A public GitHub Pages knowledge source for a Copilot agent that supports introductory Pandas CSV analysis for T Level Digital Production, Design and Development ESP students.

## Project Purpose

This repository supports a classroom Copilot agent designed to help students practise Pandas data analysis using a fresh, fictional CSV dataset.

The agent is intended to help students:

* generate a practice CSV dataset
* load CSV data using Pandas
* inspect a DataFrame
* calculate summary statistics
* use `groupby()`
* identify trends
* write evidence-based conclusions
* produce recommendations from data

The agent acts as a coach. It should guide students with hints, questions and feedback rather than simply completing the task for them.

## Teaching Context

This project is designed for:

* T Level Digital Production, Design and Development
* Employer Set Project preparation
* introductory Pandas lessons
* CSV data analysis practice
* Python data handling and interpretation
* post-16 learners with mixed confidence and prior knowledge

The resource helps students transfer the skills from a taught lesson into a new but related scenario.

## What This Repository Contains

This repository is used to host a simple public knowledge page for a Copilot agent.

The public page gives the agent guidance on:

* the type of CSV data to generate
* the Pandas commands students are expected to practise
* the common mistakes students may make
* the style of support the agent should provide
* the kind of questions the agent can ask
* how students should explain findings using evidence

## Example Practice Scenario

The practice task is based on a fictional training event.

A business, college or training provider has collected data about learners completing digital skills activities. Students use Pandas to analyse the data and make recommendations based on evidence.

The data is fictional and should not contain real student names or personal information.

## Recommended CSV File Name

Students should normally save the generated CSV file as:

```text
practice_training_data.csv
```

This keeps the file name clear and allows students to use the same starter code pattern.

## Recommended CSV Columns

The generated CSV dataset should normally include columns such as:

```text
Date
Session
Learner ID
Activity
Score
Completion Time (mins)
Hints Used
Confidence Rating
Difficulty
```

Students should be reminded that Pandas column names must match exactly.

## GitHub Pages Use

This repository is intended to be published using GitHub Pages so the HTML knowledge page can be used as a public website source for a Copilot agent.

The Copilot agent should be pointed to the published GitHub Pages URL, not to the repository editing page.

## Safeguarding and Data Use

This project must only use fictional data.

Do not include:

* real student names
* real student IDs
* personal information
* internal college data
* confidential assessment material
* private links or documents

The dataset generated by the agent should be realistic enough for practice but fully fictional.

## Intended Agent Behaviour

The agent should:

* use British English
* keep explanations clear and student-friendly
* give hints before full answers
* encourage students to run code themselves
* help students debug errors
* support evidence-based written conclusions
* avoid using advanced Pandas unless appropriate

The agent should not replace student thinking. It should support students to become more independent with Pandas analysis.
